# GitHubPage

這是一個使用 GitHub Pages 架設的靜態網站專案，包含多個 HTML 頁面、CSS 樣式表和對應的 JavaScript 檔案。  

> ⚠️ **注意**：本專案所有內容（HTML、CSS、JavaScript、README 等）皆由 GPT 協助生成。  

## 📁 專案結構

- index.html                     - 網站首頁
- Image-Paste-Downloader.html    - 圖片粘貼下載頁面
- Tag-o-Rama.html                - 標籤功能頁面
- Twitter-Page.html              - Twitter 整合頁面
- css/                           - 存放 CSS 樣式表
- js/
  - Image-Paste-Downloader.js    - 對應圖片粘貼下載頁面的 JS
  - Tag-o-Rama.js                - 對應標籤功能頁面的 JS
  - Twitter-Page.js              - 對應 Twitter 整合頁面的 JS
- README.md                      - 專案說明文件

## 🌐 使用方式

1. 克隆或下載此倉庫。  
2. 打開 `index.html` 檔案於瀏覽器中查看網站首頁。  
3. 使用其他 HTML 頁面以體驗不同功能，每個頁面會自動載入對應的 JS。  

## 🛠 技術棧

- **HTML**：網頁內容結構。  
- **CSS**：網頁樣式設計。  
- **JavaScript**：實現網頁互動功能，每個 HTML 頁面有對應的 JS 檔案。
